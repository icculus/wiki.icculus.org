# PHYSFS_VER_MINOR

The current minor version of the PhysicsFS headers.

## Header File

Defined in [<physfs.h>](https://github.com/icculus/physfs/blob/main/src/physfs.h)

## Syntax

```c
#define PHYSFS_VER_MINOR 3
```

## Remarks

If this were PhysicsFS version 3.2.1, this value would be 2.

## Version

This macro is available since PhysicsFS 1.0.0.
