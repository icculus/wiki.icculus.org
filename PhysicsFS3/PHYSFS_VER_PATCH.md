# PHYSFS_VER_PATCH

The current patch (or micro) version of the PhysicsFS headers.

## Header File

Defined in [<physfs.h>](https://github.com/icculus/physfs/blob/main/src/physfs.h)

## Syntax

```c
#define PHYSFS_VER_PATCH 0
```

## Remarks

If this were PhysicsFS version 3.2.1, this value would be 1.

## Version

This macro is available since PhysicsFS 1.0.0.
