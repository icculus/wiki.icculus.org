# CONTROLLERIMAGE_MICRO_VERSION

The current micro (or patchlevel) version of ControllerImage headers.

## Header File

Defined in [<controllerimage.h>](https://github.com/icculus/ControllerImage/blob/main/src/controllerimage.h)

## Syntax

```c
#define CONTROLLERIMAGE_MICRO_VERSION   2
```

## Remarks

If this were ControllerImage version 3.2.1, this value would be 1.

## Version

This macro is available since ControllerImage 1.0.0.
