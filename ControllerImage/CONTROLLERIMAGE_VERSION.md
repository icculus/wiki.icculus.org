# CONTROLLERIMAGE_VERSION

The current version number macro of the ControllerImage headers.

## Header File

Defined in [<controllerimage.h>](https://github.com/icculus/ControllerImage/blob/main/src/controllerimage.h)

## Syntax

```c
#define CONTROLLERIMAGE_VERSION SDL_VERSIONNUM(CONTROLLERIMAGE_MAJOR_VERSION, CONTROLLERIMAGE_MINOR_VERSION, CONTROLLERIMAGE_MICRO_VERSION)
```

## Version

This macro is available since ControllerImage 1.0.0.
