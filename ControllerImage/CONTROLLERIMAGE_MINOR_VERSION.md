# CONTROLLERIMAGE_MINOR_VERSION

The current minor version of ControllerImage headers.

## Header File

Defined in [<controllerimage.h>](https://github.com/icculus/ControllerImage/blob/main/src/controllerimage.h)

## Syntax

```c
#define CONTROLLERIMAGE_MINOR_VERSION   0
```

## Remarks

If this were ControllerImage version 3.2.1, this value would be 2.

## Version

This macro is available since ControllerImage 1.0.0.
