# SDL_sound 3.0

The latest version of this library is available from [GitHub](https://github.com/icculus/SDL_sound/releases).

This is a simple audio file decoding library for use with SDL3.
It is available under the zlib license, found in the file [LICENSE.txt](https://github.com/icculus/SDL_sound/blob/main/LICENSE.txt).
The API is listed in [CategorySDLSound](CategorySDLSound).

Enjoy!
